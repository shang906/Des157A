Learning Journal 1

By looking into the best practice of design, one broad idea that I have found is that there are many minor things that could influence the outcome of one website. In order to make a website that is user friendly and easy to read, as the designers we need to guide the users on every single step so that they can fully understand what are they being expected to be putting in when they are submitting a form. One really interesting thing that I found when I was reading the article is how there should not be a refresh button being set on the website. I would usually think that would be the first thing that I would try since it would be easier to have some feedback, but as the designer, it is important to remember how it would influence others and how it might not be the best solution.
